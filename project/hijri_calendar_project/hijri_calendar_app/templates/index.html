Rest API for Official, Cultural and Religous Holidays

This website is created to display hijri calendar and official, cultural and religeous holidays from various sources, such as pakmoonsighting.pk. The web Site offers data via publicly accessible Restful API for free.

Features

Holiday Calendar: /holiday/

Holiday Calendar displays table of public holidays observed in Americas (US/Canada) and South asian countries such as Pakistan, India and Bangladesh.

Holiday API: /api/v1/holiday/ (opens in new window)

The Holiday API provides access to religeous, Cultural and offician holidays. API is public and require no authenticatation for read-only access.

Calendar API: /api/v1/calendar/ (opens in new window)

The Calendar API provides access to two years (Y2019 and Y2020) worth of hijri calendar. API is public and require no authenticatation for read-only access.
